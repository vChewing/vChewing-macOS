# 【教學：數字快打】哪裡都是計算機、簡單輸入中文數字、各式各樣數字轉換

數字快打提供四種數字轉換，原先是為行政、會計等人員設計，實際開發後發現這個功能大家都用得到！而且評價也非常高，用過的朋友都表示數字快打真的能提升工作效率。

## 數字快打使用說明：

### ① 需要先確認有開啟數字快打功能

開啟 **偏好設定** → 切換到 **行為** 分頁 → 在分頁 **Ｂ** 內找到「**數字快打**」並確認已打勾啟用。

> 預設值為開啟。僅在大千傳統、微軟、零壹、ETen26 鍵盤排列下有效。

### ② 啟動鍵為兩下漂鍵（`` ` ``）

在輸入法處於中文組字狀態時，**連按兩下漂鍵**（鍵盤左上角的 `` ` ``，不需按 Shift）即可進入數字快打模式。

- 組字區前方若有已輸入的中文內容，**不會被清空**，數字快打結果會附加在後方一併送出。
- 若 300ms 內未按第二下，會改為叫出符號表（與未開啟數字快打時的單按漂鍵行為一致）。

## 四大子功能：

### 1、數字

按兩下漂鍵啟動數字快打後，輸入任一數字，即可帶出相關的數字符號。在 Word 打字、Email 發信、寫社群貼文時都非常方便。

輸入兩位數以上的數字時，即可帶出相關的轉換結果，包含千分位逗號、中文大小寫等等。

**以 `123456` 為例，候選清單包含：**

| 順序 | 格式 | 輸出 |
|------|------|------|
| 1 | 千分位 | `123,456` |
| 2 | 中文財務大寫（含元整） | `壹貳參仟肆佰伍拾陸元整` |
| 3 | 中文小寫完整讀法 | `十二萬三千四百五十六` |
| 4 | 混合格式 | `12萬3,456元` |
| 5 | 逐位中文小寫 | `一二三四五六` |
| 6 | 逐位中文大寫 | `壹貳參肆伍陸` |

### 2、算盤（計算機）

啟動數字快打後，可以直接輸入基本的四則運算（`+ - * /`，支援小數與括號），不需要額外開啟計算機應用程式，只要是能打字的地方都能使用。

組字區右側會即時顯示 `= <計算結果>` 預覽。

**以 `20*300` 為例，候選清單包含：**

| 輸出 |
|------|
| `6000` |
| `6,000` |
| `20*300=6000`（運算式格式） |
| `六千` |
| `陸仟` |

### 3、日期

輸入 `年.月.日`（或 `/`、`-` 分隔），即可帶出多種日期格式，不需要切換中英模式。

- 年份 ≤ 150 → 視為民國年自動換算；年份 > 150 → 視為西元年

**以 `2026.4.5` 為例，候選清單包含：**

| 輸出 |
|------|
| `2026年4月5日` |
| `民國115年4月5日` |
| `115年4月5日` |
| `2026/04/05` |
| `2026-04-05` |
| `二○二六年四月五日` |

### 4、時間

輸入含 `:` 的時間字串（24 小時制），即可帶出多種時間格式。

**以 `14:30` 為例，候選清單包含：**

| 輸出 |
|------|
| `14:30` |
| `下午2點30分` |
| `下午2:30` |
| `14時30分` |
| `十四點三十分` |

## 操作按鍵總覽

| 按鍵 | 行為 |
|------|------|
| ↑ / ↓ / ← / → | 在候選窗中移動選取 |
| **Enter** | 確認選取的候選（含前方中文）|
| **Space** | 確認第一個候選 |
| **Escape** | 取消，回到中文組字狀態（前方中文保留）|
| **Backspace** | 刪除最後一個字元；緩衝區空時退出模式 |
