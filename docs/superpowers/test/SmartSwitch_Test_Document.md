## 測試重點

1. **連續輸入測試**：輸入 "mail"、"app"、"Google" 等英文單字時，應該在第 2 個字母時自動切換為英文模式

2. **返回中文測試**：
   - 按 **空白鍵** 提交英文並返回中文
   - 按 **Tab 鍵** 提交英文並返回中文
   - 按 **Enter 鍵** 提交英文並返回中文（Enter 被消耗，不穿透給應用程式；避免意外送出表單或換行）
   - 輸入 **標點符號**（如 ? 或 ，）提交英文並返回中文
   - 連按 **Backspace 兩次** 刪除英文並返回中文

3. **連續輸入測試**：
   - 輸入 "test" 應該顯示 "test"（不是 "est"）
   - 輸入 "hello" 應該顯示 "hello"

## 常見問題排除

如果智慧中英切換無法正常運作：

1. **確認已啟用**：偏好設定 → 行為設定 → 勾選「智慧中英文切換」
2. **確認鍵盤排列**：此功能主要針對大千注音排列優化
3. **檢查觸發條件**：需要連續 2 個按鍵無法組成有效注音讀音才會觸發
